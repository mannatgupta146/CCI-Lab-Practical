This repository contains a comprehensive, single-page HTML study guide for the CCI (Computer Crime Investigation) lab practical exam. It's designed as a "study hack" to consolidate all 10 practicals into one easy-to-use, responsive file.

This guide was built to be a fast, all-in-one resource for last-minute revision, with all procedures, tools, and key concepts in one place.

### How to Enable GitHub Pages:
1.  Go to your repository's **Settings**.
2.  Click on **Pages** in the left sidebar.
3.  Under "Build and deployment," set the **Source** to **"Deploy from a branch"**.
4.  Select your `main` branch, keep `/root` as the folder, and click **Save**.
5.  Wait a minute for it to deploy, then visit the link above!
